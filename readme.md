# Установка
### Required Components

- Node.js
    - <http://nodejs.org/>
    - v4.4.x is recommended.

## Set Up

#### 1) Install gulp:

*Note:* If you have already installed gulp, skip this section.

    $ npm install -g gulp

#### 2) Clone Rin:

    $ git clone git@github.com:sanographix/rin.git test-repo
    $ cd test-repo
    $ npm install

#### 3) Run gulp:

    $ gulp
